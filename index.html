Traffic Camera Tracking

Research Project at CARISSMA, Ingolstadt

Opensource project to detect, track and analyze VRU's (Vulnerable Road Users) at a Traffic Intersection. Main contributor in this project as a part-time developer at CARISSMA Research Institute Github Link

Hand-labelled thousands of images for training data and trained a YoloV5 neural network for detecton followed by SORT Algorithm for tracking. Using real-world to camera homography, extracting the 3D world position of the tracker and using it for velocity estimation. OCR Scripts implemented to extract date-timestamp and accelerated inference using TensorRT
